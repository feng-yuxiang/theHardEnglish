## introduce  
恍然大悟，其幸哉？其悲哉？  
人生若没有几个恍然大悟，其可想而知。  
同样的笔，在诗人和画家手中是不是截然不同？语言也是如此。  
司空见惯，但语言不但是人们认识世界的产物，也是改造世界的工具，其之所含，岂止天地？  
掌握一门外语，你会体验到挣脱母语的自由，岂止只是一个新世界。  
如果你来到这里，那么请和我们一起交流体验： 生活不是缺少美，而是缺少发现。
